Employee Management System

Overview:
Microservices-based application developed using Java and Spring Boot to streamline employee record management.

Key Features:

  • Employee registration, update, delete & search
  • Role-based access control (Admin/User)
  • RESTful APIs for communication
  • Centralized MySQL database
  • Reporting module for salary insights

Technologies Used:

Java • Spring Boot • Hibernate/JPA • Spring Data • MySQL • Spring Security • Docker

Architecture:

Employee Service • Department Service • Authentication Service
